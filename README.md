# shell_script

### File Extension:

```
<filename>.sh
```

#### 1) Listing File Permissions
```
ls -l
```
#### Permissions

```
1 execute
2 write
4 read
7 read write execute
777 means read write execute for owner(7),groups(7),others(7)
```

#### 2) Give read, write, and execute permissions for everyone
```
chmod 777 <filename>.sh
```
